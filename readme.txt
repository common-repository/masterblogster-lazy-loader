== Description ==

Bored with the static home page of your blog? Masterblogster Lazy Loader plugin loads your blog posts one by one with cool sliding animation effect. This plugin works with HTML5 valid wordpress blogs as it utilizes <article> tag to animate the blog posts.

Here is the live demo http://www.masterblogster.com

Check out plugin home page for more details at http://www.masterblogster.com/plugins/masterblogster-lazy-loader

This plugin is used in top sites like http://techgyo.com

[youtube http://www.youtube.com/watch?v=RKk_yk5XKIQ]

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
